## 課題1のプログラムの内容
入力したn(10進数)をそれぞれ2進数、8進数、16進数に変換した値を出力し、合計値も同様に出力する。

## クローン方法
このリポジトリをターミナルで下記のようにクローンしてください。
``` 
$ git clone git@github.com:peke0157/robosys2025.git
```

## 必要なソフトウェア
- Python 
    - テスト済みバージョン3.7~3.10

## テスト環境
